# Mekanism Enchantable

Mekanism stuffs are now enchantable!

## Features
- Makes these items enchantable by Enchanting Table
  - Atomic Disassembler
  - Electric Bow
  - Meka-Tool
  - Jetpack / Armored Jetpack
  - Free Runners / Armored Free Runners
  - Hazmat Suit
  - Scuba Mask / Scuba Tank
  - MekaSuit
